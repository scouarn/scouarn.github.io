Mode 2D et refactorisation 01/03/2020

Je souhaite que les joueurs puissent choisir entre la vue en 3D et une vue en 2D type "radar". J'ai ajouté cette fonctionnalité en réutilisant l'algorithme de lancer de rayons utilisé pour le rendu 3D. J'en ai profité pour refactoriser le code et ajouter un menu.

Les contrôles sont différents entre les deux vues mais je compte aller plus loin en permettant aux joueurs de choisir définir tous ces paramètres en construisant le "robot" qu'ils contrôlent. Ansi un robot avec des chenilles aura des contrôles différents d'un robot qui vole; un robot avec une caméra n'aura pas la même vue qu'un robot avec un capteur de distance; etc...

La touche échap permet de revenir au menu une fois en jeu.
