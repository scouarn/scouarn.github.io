Jeu de labyrinthe 22/02/2020

J'ai créé un système de physique pour le déplacement d'un personnage.
J'ai réutilisé le code du labyrinthe, rejoignez le carreau vert. Attention à l'inertie !

Contrôles :

  • Z : avancer
  • S : reculer
  • Q : déplacement gauche
  • D : déplacement droite
  • ← : tourner à gauche
  • → : tourner à droite
